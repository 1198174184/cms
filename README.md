## Requirement

1. PHP >= 5.5.9
2. composer
3. node

## Installation

```shell
git clone https://github.com/Bright-Tech/cms.git
```

```shell
composer install
```

```shell
php artisan migrate
```

```shell
php artisan passport:install
```
